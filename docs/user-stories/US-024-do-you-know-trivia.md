# US-024: Do You Know? Trivia & Interesting Facts Page

## Story

As a **visitor**,
I want to view a "Do You Know?" page with interesting facts, records, and trivia about the tournament,
so that I can learn fun details and be entertained while exploring the site.

## Context

- The "Do You Know?" section highlights fun facts, records, and trivia related to Ölcupen's history and culture.
- Examples: longest winning streak, highest scoring match, oldest participant, funny tournament moments, etc.
- This content humanizes the tournament and provides light entertainment for fans.
- Data can be hardcoded initially (as a static page), later enhanced with dynamic content and user interactions.
- This is Phase 2a work: a new static page to add personality and engagement to the site.

## Acceptance criteria

- [ ] Given I navigate to the "Do You Know?" page, when the page loads, then I see a collection of interesting facts and trivia.
- [ ] Given I view the trivia section, when I read the facts, then each fact is presented clearly with a title and a brief description (1-3 sentences).
- [ ] Given I view multiple facts, when I scroll through them, then they are presented in an engaging layout (cards, callout boxes, or an accordion-style list).
- [ ] Given I'm on the trivia page, when I view the navigation, then there's a clear link from the main menu or footer to reach this page.
- [ ] Given I view a fact on a mobile device, when I read it, then the text is readable and the layout is responsive (no horizontal scrolling).
- [ ] Given I view the facts, when I see them, then there's visual variety (icons, colors, or imagery) to keep the page engaging and less text-heavy.

## Notes

- Facts can include records (e.g., "Most goals in a single match: 12"), memorable moments, or humorous observations.
- Consider using icons or emojis to visually categorize facts (⚽ for goals, 🏆 for records, 😂 for funny moments, etc.).
- This page can later include sorting/filtering by category or a random fact feature using JavaScript.
- Design should feel playful and inviting to match the tournament's fun, casual vibe.
